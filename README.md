### ✔알고팜(Algo Farm)  
- 알고팜은 Algorithm(알고리즘) + Farm(농장)을 결합한 뜻으로, 귀여운 캐릭터와 상호작용하며 알고리즘 스터디의 참여율을 높이고자 합니다.

### ✔기획배경
```
개발자 채용시장이 점차 어려워지는 만큼, 코딩테스트의 중요도가 높아지고 있지만, 
지속적으로 알고리즘 문제를 푸는 것은 쉽지 않습니다.
```
